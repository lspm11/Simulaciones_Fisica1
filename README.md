# Simulación Ejercicio 6 - Práctico 4

Proyecto React + Vite listo para subir a GitHub y publicar luego en Vercel.

## Requisitos

- Node.js instalado
- npm

## Instalación

```bash
npm install
```

## Ejecutar en local

```bash
npm run dev
```

## Generar versión de producción

```bash
npm run build
```

## Archivos principales

- `src/SimulacionPoleaIdeal.jsx`: simulación
- `src/App.jsx`: componente principal
- `src/styles.css`: estilos

## Subir a GitHub

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos de esta carpeta.
3. Luego conectar el repositorio con Vercel para publicarlo.
